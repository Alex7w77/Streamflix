STREAMFLIX - Sistema de Recomendacion con IA (Java)
====================================================

COMPILAR (desde la carpeta del proyecto):
  javac -d bin src/streamflix/*.java src/streamflix/*/*.java

EJECUTAR:
  java -cp bin streamflix.StreamFlixMain

Requisitos: Java 17 o superior (probado con Java 21).

NOTAS DE LA VERSION 2.0
-----------------------
CODIGO
- Filtrado colaborativo con PREDICCION CENTRADA EN LA MEDIA
  (pred = media_usuario + suma ponderada de desviaciones de los vecinos).
  Resultado: MAE 1.013 -> 0.913 ; F1 0.092 -> 0.147 ; Precision@5 0.056 -> 0.096.
- Mayor densidad de datos (686 calificaciones) y K=8 vecinos.
- Nueva clase util/ConsoleUI.java: banners, secciones, tablas con bordes y
  colores ANSI (se desactivan automaticamente al redirigir la salida).
- Toda la salida usa ASCII, por lo que se ve correcta en la consola de Windows.
- Javadoc ampliado en las clases nuevas y modificadas.

INFORME
- Diseno moderno (portada tipo "hero", chips de metricas, tablas cebra,
  cajas de resalte) y 7 graficos generados a partir de los datos reales.

ARCHIVOS
- src/                 codigo fuente Java
- SALIDA_CONSOLA.txt   ejemplo de ejecucion completa
- charts/              graficos del informe (PNG)
- make_charts.py       script que genera los graficos
